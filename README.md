# reg-exp-practice 正则表达式 练习

## 系统技术方案
- 前端 vue.js
- 后端 node.js - koa.js
- 数据库 redis - 持久化

## 开发模式
- 前后端分离开发，后端只管吐index.html
- 单页应用，前端history路由

## 运行
- client端
- cd client
- npm i
- npm run serve
- server端
- cd server
- npm i
- npm run dev
- 浏览器访问 0.0.0.0:3000
